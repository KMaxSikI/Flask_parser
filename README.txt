=================
Веб-сайт на Flask
=================

Этот код представляет веб-приложение на основе Flask, которое обрабатывает запросы пользователей
для поиска и парсинга товаров на внешнем ресурсе wildberries.ru по заданной категории.
Программа также сохраняет результаты парсинга в базу данных SQLite для последующего использования.

Ниже представленно краткое описание действий кода:

1. Настройка базы данных и модели товаров:
==========================================

* Используется SQLAlchemy для работы с базой данных SQLite.
* Создается модель данных Product, определяющая поля для хранения информации о товарах
  (категория, артикул, название, рейтинг, количество отзывов, цена, URL адрес).

2. Определение маршрутов Flask:
===============================

* '/' - Главная страница с описанием возможностей.
* '/contacts/' - Страница с контактными данными.
* '/form/' - Страница с формой для ввода категории товаров.
* '/results/<category>' -  Страница результатов поиска товаров по указанной категории.
* '/saved_results/' - Страница с сохраненными результатами из базы данных.

3. Обработка запросов:
=====================

* При запросе категории товаров, приложение обращается к внешнему ресурсу (wildberries.ru)
для получения информации о товарах этой категории.

* Полученные данные парсятся и сохраняются в базу данных.
* Результаты поиска отображаются на странице '/results/<category>'
  и предоставляют информацию о товарах, их рейтинге, отзывах и цене.
* Для многопоточности используется threading.Thread, чтобы сохранение данных в базу происходило асинхронно.

4. Отображение сохраненных результатов:
======================================

* '/saved_results/' позволяет просматривать сохраненные результаты из базы данных.

5. Запуск приложения:
====================

* Приложение запускается в режиме отладки через app.run(debug=True).

В целом, приложение предназначено для поиска и сохранения информации о товарах из указанной категории,
а также предоставляет интерфейс для просмотра сохраненных результатов.
